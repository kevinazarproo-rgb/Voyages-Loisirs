# Voyages Loisirs — Site vitrine

Site de **Voyages Loisirs**, agence de voyages française spécialisée dans les circuits accompagnés haut de gamme (6 destinations par an, paiement en 4×, 6× ou 10× sans frais).

## Structure

```
index.html        Page d'accueil
css/style.css     Styles (design tokens, sections, responsive)
js/main.js        Interactions (header au scroll, animations, newsletter)
docs/design/      Maquette et spécifications de référence
```

## Site 100 % statique

Aucune installation nécessaire : ouvrez `index.html` dans un navigateur, ou hébergez le dossier tel quel (GitHub Pages, Netlify…).
